Подробное описание

template<typename T, T empty_val, size_t N = 2>
class mat::matrix< T, empty_val, N >

Шаблон N-мерной матрицы.

Параметры шаблона
T- тип элемента матрицы.
empty_val- значение пустого элемента матрицы.
N- размерность матрицы (по умолчанию 2).
